诺基亚通信有限公司东莞分公司成立于1995年，历经近20年的发展，公司已成为诺基亚全球最重要的手机生产基地之一。凭借高效灵活的运作、完善的产品品质、极具竞争力的成本控制，诺基亚通信有限公司东莞分公司成为诺基亚内部的成功典范。 2006年10月，诺基亚通信有限公司东莞分公司增资扩建二期厂房投产运营后，东莞手机工厂产能的增加进一步提高了其在全球的市场竞争力，也进一步加强了诺基亚对中国的承诺，创造更多的就业机会，为促进地方经济做出了更大贡献。

工作情况 Default

岗位要求：男女不限，年龄18周岁以上，高中及以上学历，2年或以上大型制造业生产操作工作经验，具备初级英语阅读能力及计算机操作能力，能适应夜班工作。 岗位职责：负责产品的装配和包装工作；贯彻理解公司品质政策，做到不接受、不生产坏品；根据生产需要，提出合理化建议，及时向拉长汇报；严格遵守公司各项方针政策和管理制度。

要求 Default

底薪1850元+加班费（按1850的基数计算）+夜班津贴（212-239元/月）=月综合收入2900元左右。

包吃包住吗 Default

1、两班倒工作制，上班时间为四三三四（即上四天休三天，上三天休四天），每月加班时间不超过36小时。 2、为每位员工购买社会保险、住房公积金、商业保险。 3、包吃住（公司将钱补到饭卡中，自己刷卡吃饭。住宿有班车接送上下班，不收取任何费用）。

待遇怎么样 Default
